# PI的用法和意思 #

PI是圆周率的意思，就是周长与直径的比值。 周长为360度，周长简称弧长， 直接为2R。

一个圆由2个弧度组成，然后弧度是由弧长除以半径得到的值，弧度＝弧长/半径，一个弧度得到一个近似值，因此PI就是代表一个弧度的近似值，PI取值3.1415~。
那么2个弧度组成一个人圆，假设每一个弧度为n，那么2n就是一个360度角的圆，那么360度 = 2n。
每一个弧度代表180度的半圆。180° = n，那么每一角度为 = n/180°,等价于Math.PI / 180。
于是每一个角度值设为KO，那么现在角度是30度的角，就是30KO。

使用场景在计算圆周长、圆面积、球体积等几何形状的关键值。
